# loan-system-prototype

Prototipo navegable del **Sistema de Préstamos Bancarios**, desarrollado como parte del diseño de UI/UX y requisitos del sistema.

---

## 📌 Descripción

Este repositorio contiene un **prototipo funcional en React** que simula la navegación principal del sistema de préstamos, incluyendo:

- Inicio de sesión
- Dashboard
- Registro de solicitud de préstamo
- Evaluación y aprobación
- Registro de pagos
- Gestión de mora

El prototipo está alineado con las **historias de usuario** y el **flujo del ciclo de vida del préstamo**.
